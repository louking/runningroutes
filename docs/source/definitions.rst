.. _glossary:

Definitions
===============

.. glossary::
    :sorted:

    route
        describes a path for running, biking, hiking, etc.

    interest
        :term:`routes <route>` are grouped by "interest". The interest can be a running club, etc. Users
        are allowed to see all "public" interests, and if logged into **routetility** may be allowed to
        see some "private" interests as well.

    route admin
        A route admin can create and edit :term:`routes <route>` for one or more
        :term:`interests <interest>`, as set up by a :term:`super admin`.

    super admin
        A super admin can create users, create :term:`interests <interest>`, assign user roles,
        :term:`interests <interest>`, etc.
